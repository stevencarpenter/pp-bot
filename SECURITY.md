# Security Policy

## Supported Versions

Security updates are applied to the `main` branch. Older tags may not receive fixes.

## Reporting a Vulnerability

Please report security issues via GitHub Security Advisories for this repository.
Do not email maintainers. If you are unsure whether your report is security-sensitive,
open a private security advisory and include reproduction steps and impact details.

We will acknowledge receipt within a reasonable timeframe and provide updates as we investigate.
